===

2. 添加一个节点, 需要重建的索引数据约等于 总数 / 节点数
3. 一个节点失效, 需要重建的缓存数量还不知道怎么算
4. 如何处理添加节点时原节点的冗余数据，目前时等待自然失效，失效时间过长且节点频繁变动将导致获取到老数据
